## Button Details

### Button Dimensions
- **A-**: 40dp Ã— 36dp, 10sp text
- **A**: 40dp Ã— 36dp, 12sp text
- **A+**: 40dp Ã— 36dp, 14sp text
- **A++**: 44dp Ã— 36dp, 16sp text

### Button Colors
- **Active**: #0061A4 (Material Blue)
- **Inactive**: #73777F (Material Gray)

### Button Spacing
- Margin between buttons: 4dp
- Label to first button: 8dp
- Buttons to language button: space_between (fills remaining space)

## Light Mode vs Dark Mode

### Light Mode
```
Background: #F4F6F8 (Light Gray)
Text: #1C2730 (Dark Gray)
Active Button: #0061A4 (Blue)
Inactive Button: #73777F (Gray)
```

### Dark Mode
```
Background: #36393C (Dark Gray)
Text: #D3D9DE (Light Gray)
Active Button: #0061A4 (Blue)
Inactive Button: #73777F (Gray)
```

Note: Button colors remain the same in both modes for consistency.
